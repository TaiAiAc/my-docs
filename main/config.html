项目配置 #

在 .vitepress 目录下创建 config.js 文件,该文件为全局配置,与 vue-press 很相似,很多官方文档没有提到的配置建议去翻阅 vue-press 文档查找配置.

项目基础配置 #

直接上代码

module.exports = {
  title: 'vite-press入门使用', // 网站标题
  description: 'vite-press', //网站描述
  base: '/', //  部署时的路径 默认 /  可以使用二级地址 /base/
  lang: 'en-US', //语言
  // 相当于html中<head>标签
  head: [
    [
      'link',
      {
        rel: 'icon',
        href: 'favicon.ico' // 图片放在public文件夹下
      }
    ]
  ]
}

这是绿色

需要手动配置的很少,更多需求自行百度.

项目主题配置 #

module.exports = {
  ...
  // 主题配置
  themeConfig: {
    // nav 左侧logo
    logo: '/favicon.ico',
    //   头部导航
    nav: [{ text: '关于', link: '/about/' }],
  }
}

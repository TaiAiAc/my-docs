开始 #

首先项目创建老三样

1.创建文件夹

2.装包下依赖

  yarn init -y
  yarn add vitepress
  mkdir && echo '# Hello VitePress' > docs/index.md

3.配置命令行 启动和打包命令

"scripts": {
  "docs:dev": "vitepress dev docs",
  "docs:build": "vitepress build docs",
  "docs:serve": "vitepress serve docs"
}

手动创建和配置目录结构 #

约定大于配置,按官方约定创建指定的目录和文件

...
├─ docs 项目入口目录
│  ├─ .vitepress    配置项目录
|  |   ├─ dist      打包后的静态文件
│  |   ├─ config.js 项目整体配置
│  ├─ public        不会被打包的资源
│  ├─ img           会被打包的资源
│  ├─ page1         
│  ├─ page2           
│  └─ index.md      内容
└─ package.json

这时候项目基本创建完成可以运行dev康康了.
